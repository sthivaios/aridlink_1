# AridLink 1

*Note: AridLink is still in extremely early stages of development.*

AridLink 1 is an OSS ESP32-based IoT irrigation controller system. It allows the user to configure deployed irrigation controllers remotely, either from a friendly UI or programmatically.

It has been designed for applications where the irrigation schedule might have to be reconfigured depending on factors such as weather conditions or plant growth. AridLink bypasses the need to reconfigure the controller in-person, which could be somewhere very remote.

AridLink 1 is designed to work in places with LTE coverage, and allows for easy deployment as it does not require the deployment of a base station. Additionally, it can be powered from any power source, including solar panels.
